About this Theme

This is a theme for Jekyll, made by Igor Padoim, which you can access (and fork and star ;) ) at its repository.

It is not intended for blogs, however. This theme tweaks on the Jekyll engine to create choose-your-own-adventure games, with the original intent and design aimed towards tabletop RPGs (D&D, I'm looking at you). Usage instructions for the theme (not this adventure) can also be found at the repository.
